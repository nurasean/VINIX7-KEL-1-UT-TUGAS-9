# Tugas Minggu 9 - 2ndTime Marketplace

Project Next.js untuk tugas minggu 9.

## Fitur
- Middleware untuk proteksi dashboard
- Zod Validation
- URL as State untuk pencarian
- Optimistic UI untuk hapus produk checkout
- Loading Skeleton

## Cara Menjalankan

npm install
npm run dev

Buka:
http://localhost:3000

## Environment

Buat file .env.local berdasarkan .env.example.
